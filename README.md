# claude/grupos

Landing page + quiz de 5 etapas para o método **Grupo 10·1** — acesso ao plano Claude Max 20x dividido em 10 pessoas por R$ 110/mês.

## Arquivos

- `index.html` — landing page principal (hero, como funciona, FAQ, CTA)
- `quiz.html` — quiz de qualificação em 5 etapas + tela de confirmação com CTA no WhatsApp
- `mascot.js` — NPC pixel-art que pula pelo viewport, pisca, reage a cliques e hover
- `logo.svg` — logo starburst terracotta

## Rodar localmente

Abra `index.html` no navegador. Sem build, sem dependências.
